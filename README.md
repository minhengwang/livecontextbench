# LiveContextBench — website

Static site for **LiveContextBench**, a benchmark for live context management in AI agents.

- `index.html` — landing page: tasks, harnesses, leaderboard, per-task write-ups
- `visualization.html` — turn-by-turn replays of real agent runs (4 tasks × 4 methods), with per-run animated summaries and an A/B compare mode

Everything is self-contained single-file HTML (no external assets).
